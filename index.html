How It Works

  1. Enter a Discord message you want to send
  2. Select the date and time to send it
  3. Click "Schedule Message" to save it
  4. The message will be sent automatically at the scheduled time
  5. Messages are stored in GitHub and sent using GitHub Actions

Note: The GitHub token needs the repo scope for this to work properly.

GitHub Token Required

This app uses GitHub to store scheduled messages. Please enter a GitHub Personal Access Token with repo scope:

You can create a token at: GitHub Settings → Developer Settings → Personal Access Tokens

Make sure to select the repo scope when creating the token.
